Unix 专有服务¶

本章描述的模块提供了 Unix 操作系统独有特性的接口，在某些情况下也适用于它的某些或许多衍生版。 以下为模块概览：

  • posix --- 最常见的 POSIX 系统调用
    • 大文件支持
    • 重要的模块内容
  • pwd --- 用户密码数据库
  • spwd --- The shadow password database
  • grp --- 组数据库
  • crypt —— 检查 Unix 口令的函数

    • 模块属性
    • 模块函数
    • 示例
  • termios --- POSIX 风格的 tty 控制

  • fcntl —— 系统调用 fcntl 和 ioctl
